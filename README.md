# Connect four game

## Игра "Четыре в ряд"
- Реализовать игру 4 в ряд для двух игроков, в двух разных терминалах, используя bash

## Запуск (находясь в директории с bash-скриптами):
- Для первого терминала (игрок 1):
    ```bash
    ./player1
    ```
- Для второго терминала (игрок 2):
    ```bash
    ./player1
    ```
